#### 문제 설명
- 문자열 str이 주어질 때, str을 출력하는 코드를 작성해 보세요.

#### 제한사항
- 1 ≤ str의 길이 ≤ 1,000,000
- str에는 공백이 없으며, 첫째 줄에 한 줄로만 주어집니다.
  
#### 입출력 예<br>
*입력 #1*<br>
**HelloWorld!**<br>
*출력 #1*<br>
**HelloWorld!**

#### 정답
```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String a = sc.next();
        System.out.println(a);
    }
}
```
